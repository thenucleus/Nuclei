AssertExtensions

EqualityContractVerifier<T>

The typed-base class for contract verifiers that test if the equality contract is implemented correctly.

EqualityContractVerifierTest

The base class for tests that verify that object.Equals(object), IEquatable{T}.Equals(T) and the == and != operators are implemented correctly.

ExceptionContractVerifier<TException>

Defines the base class for tests that verify that exceptions are correctly implemented.

HashCodeContractVerifier

The base class for contract verifiers that test if object.GetHashcode() is correctly implemented.

HashCodeContractVerifierTest

The base class for tests that need to verify that object.GetHashcode() is implemented correctly.

NotEnoughHashesException

An exception thrown if not enough hashcodes are provided to the HashCodeContractVerifier.
